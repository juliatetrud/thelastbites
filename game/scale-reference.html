Architecture

Structural room elements. Wall panels and floor bands are tiling fills; door is discrete.

Fixtures

Wall-mounted elements. All share mounting height y ≈ 58 from room top (door-top height).

Furniture

Movable or scene-fixed objects.

Props (interactable)

Objects Pip can inspect, carry, or interact with.

Characters

All adult NPCs render in the 60–72 px band; specific characters vary in silhouette but scale is consistent. Orange border = current implementation does not match canonical scale.

UI Elements

HTML overlays, not canvas-drawn. Shown at representative scale.

Echo-creatures (per chapter)

Canonical scale: 0.06–0.14 H (7–16 px). Entries marked [DISCREPANCY] exceed this in art-asset-list.md.

How to read this chart

Each cell is a 140×140 px canvas. The faint warm-brown rectangle on the left is always the reference door: 32 px wide, 110 px tall. Its top edge is always at y=10; its bottom at y=120 (the floor line). The component is drawn to the right, with its floor at y=120. A dashed vertical line marks the door's full height for easy comparison.

[PROPOSED] = scale set by this document; not yet confirmed by the lead designer.
Orange border = implementation in game/index.html differs from the canonical scale shown here.
