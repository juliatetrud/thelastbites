# The Last Bites

A narrative side-scrolling adventure game for the recipe site. Pip, a young ghost-boy on a haunted cruise ship, tastes his way through regional cuisines and folklore so he can move on.

## Project structure

- `design-docs/` — canonical design package (story bible, game design, art, UI spec, tech architecture, roadmap). Read these first.
- `chapter-specs/` — per-chapter outlines and detail specs.
- `sprints/` — sprint specs for Claude Code to execute against.
- `prototype/` — reference-only original prototype.
- `game/` — production game (built sprint by sprint).
- `bootstrap-prompt.md` — paste this as the first message to Claude Code.
- `github-structure.md` — epic and issue organization for tracking work.
